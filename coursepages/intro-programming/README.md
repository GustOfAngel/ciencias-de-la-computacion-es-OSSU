## CS50P: Introducción a la Programación con Python

Este curso ha sido desarrollado por el equipo de CS50 en la Universidad de Harvard.
> Una introducción a la programación utilizando un lenguaje llamado Python. Aprende a leer y escribir código, así como a probarlo y "depurarlo". Diseñado para estudiantes con o sin experiencia previa en programación que deseen aprender Python específicamente. Aprende sobre funciones, argumentos y valores de retorno (¡qué maravilla!); variables y tipos; condicionales y expresiones booleanas; y bucles. Aprende a manejar excepciones, encontrar y corregir errores, y escribir pruebas unitarias; usar bibliotecas de terceros; validar y extraer datos con expresiones regulares; modelar entidades del mundo real con clases, objetos, métodos y propiedades; y leer y escribir archivos. Oportunidades prácticas para mucha práctica. Ejercicios inspirados en problemas de programación del mundo real. No se requiere software excepto un navegador web, o puedes escribir código en tu propia PC o Mac.

**Enlace**: <https://cs50.harvard.edu/python/>

**Nota**: Este curso es *diferente* de CS50 o CS50x. CS50 no forma parte del plan de estudios de OSSU. Dicho esto, si has completado CS50, puedes omitir este curso y pasar al siguiente.

### Correcciones

1. Si estás haciendo la lección de BeautifulSoup4, hay un problema con Python 3.10+ que te dará un error haciendo referencia a la biblioteca `Collections`. Tenemos una solución para ti. No esperamos que la entiendas, solo pon esto al principio de tu código en el bloque de importaciones:

```python
import collections
collections.Callable = collections.abc.Callable

from bs4 import BeautifulSoup 
```

Hacer esto debería solucionar el problema de compatibilidad y permitir que tu código se ejecute.
